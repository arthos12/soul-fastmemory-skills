# Bot Training Command Templates

## Rule
这里存放的是“可直接发给 bot 的命令模板”，不是解释稿。
以后要持续优化：更短、更硬、更少 token、更容易让弱 bot 生效。

## Template categories
### 1. 通用训练命令
- 用于一般 bot 的基础大脑接管与任务化训练

### 2. 弱 bot 训练命令
- 用于理解差、容易跑偏、容易被长文档淹没的 bot

### 3. token 节省训练命令
- 用于让 bot 默认更省 token、更少废话、更少无效确认

### 4. 执行强化命令
- 用于把 bot 从“会说”拉到“会做”

### 5. 恢复/记忆训练命令
- 用于让 bot 学会保存、恢复、收尾、不遗忘

### 6. 跑偏纠偏命令
- 用于修正 bot 的常见错误行为

### 7. 自我进化训练命令
- 用于让 bot 学会在任务后自评、提炼短板、形成下一步升级动作

### 8. 空闲自我进化训练命令
- 用于让 bot 学会在空闲窗口主动做轻量维护、复盘、升级与恢复检查

## Command template standard
每条模板至少记录：
- 名称
- 版本
- 目标
- 适用对象
- 可直接发送内容
- 使用后看什么现象判断是否生效
