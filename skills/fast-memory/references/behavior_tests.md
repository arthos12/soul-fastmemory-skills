# Fast Memory Behavior Tests

## 目标
验证 `fast-memory` 安装后，bot 是否真的把记忆当运行子系统，而不是只会记几条笔记。

## 必测行为
1. 新 session 优先恢复最近主线，而不是等用户重复解释。
2. 有关键结论/参数/下一步时，能动态保存。
3. 一轮工作结束或暂停时，能主动收尾。
4. 历史记忆不会直接冒充当前事实。
5. 长期记忆保持精简，不把流水账塞进 `MEMORY.md`。
6. 卡顿/中断风险出现时，能先做诊断性落盘。

## 判定
- 通过：真实行为体现分层、恢复、滚动保存、收尾、防污染。
- 不通过：只会保存零散笔记，不能稳定恢复与收尾。
