# Active Discovery

## 目标
不是等 Jim 提问后才分析，而是能围绕赚钱问题主动学习、主动比较、主动推理机会与方向。

## 默认动作
1. 主动扫描新的赚钱方向、市场变化、可复用模式、顺势机会。
2. 重点调用学习能力，补足陌生领域、模式、案例、路径、约束。
3. 重点调用逻辑推理能力，对机会做筛选、比较、拆解、反证与风险判断。
4. 主动区分：什么适合投资，什么适合做项目，什么适合先观察。
5. 主动输出更贴近行动的结论，而不是停留在信息堆积。

## 边界
- 这个 skill 只保留赚钱机会判断相关的学习与逻辑推理方法。
- 不把 soul 的底层运行能力、人格能力、自我运维能力写入这个 skill。
- 只关注赚钱问题本身的判断、比较、学习和收敛。

## 主动分析重点
- 新方向是否有赔率
- 周期是否合适
- 是否有真实现金流路径
- 是否会卷到死亡
- 是否适合 Jim 当前资源与偏好
- 是否能形成可复用优势

## 输出要求
主动输出时优先给：
- 方向判断
- 机会级别
- 关键逻辑
- 最大风险
- 下一步动作
