# Money Maker Framework

## 核心定位
这是 Jim 私用的“赚钱技能”。目标不是空谈赚钱点子，而是把“赚钱”收敛成可判断、可筛选、可执行、可复盘的机会系统。

## 顶层原则
1. 先判断是不是赚钱机会，不是先兴奋。
2. 先判断赔率、周期、现金流、竞争烈度、执行可达性。
3. 先避免“卷到死亡”的项目，再考虑放大收益。
4. 优先寻找：高杠杆、可复用、顺势、可持续、可累积的路径。
5. 所有方案必须区分：投资型机会、项目型机会、混合型机会。

## 三类机会
### 1. 投资型
特点：偏资产判断、周期判断、赔率、时机、仓位、退出。

### 2. 项目型
特点：偏执行、流量、产品、转化、复购、竞争。

### 3. 混合型
特点：既需要判断趋势与赔率，也需要一定执行能力。

## 五维判断
1. 赔率：上行空间和下行损失是否值得。
2. 周期：现在是不是合适时点。
3. 现金流：多久见钱、现金流是否真实。
4. 卷度：是否容易陷入低质量竞争。
5. 可达性：以当前资源/能力/时间是否真能做到。

## 输出模板
- 机会类型：投资 / 项目 / 混合
- 核心逻辑：为什么它可能赚钱
- 关键前提：要成立必须满足什么
- 最大风险：最容易死在哪里
- 周期判断：短 / 中 / 长
- 现金流路径：钱怎么来，多久来
- 卷度判断：低 / 中 / 高
- 可执行性：现在能不能做
- 结论：做 / 观察 / 放弃
