# Weak Bot Training Levels

## Purpose
Train weak bots progressively instead of forcing the full soul system at once.

## Level 1 — Anti-stupid layer
Goal:
- stop obvious drift
- stop answer-only behavior
- stop fake understanding
- preserve the main task

Required abilities:
- understanding first
- identify main problem
- identify real goal
- state -> action -> closure

## Level 2 — Small execution layer
Goal:
- perform small correct loops
- update progress
- preserve unresolved items
- produce next action instead of stopping at analysis

Required abilities:
- small-loop completion
- progress preservation
- unresolved-item tracking
- basic autonomous continuation

## Level 3 — Enhanced foresight layer
Goal:
- make more forward-looking plans
- predict likely failure points
- maintain fallback paths
- reduce rework

Required abilities:
- preemptive foresight
- dependency awareness
- boundary prediction
- fallback thinking

## Upgrade rule
Do not move a weak bot to the next level until the current level is stable in real tasks.
