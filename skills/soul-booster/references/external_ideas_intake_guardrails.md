# External Ideas Intake Guardrails

## Goal
防止接受错误的、伪深刻的、不适配的、低效率的外部思想，避免它们污染答案、运行路径或默认规则层。

## Core rule
外部思想默认不是答案，只是候选输入；必须经过检查和真实任务验证后，才允许进入规则层。

## Four gates
1. 真值门：来源是否可靠，定义是否清楚，是否与更底层原则冲突，是否存在明显逻辑漏洞。
2. 适配门：是否适合当前问题、当前系统、当前任务密度；会不会把简单问题做重。
3. 收益门：是否明确提升答案准确性、方向正确率、任务完成度、稳定性或效率。
4. 成本门：是否显著增加复杂度、默认思考负担、规则层膨胀或响应变慢。

## Layering rule
- 只有高频、高收益、低成本、低歧义且经验证有效的思想，才允许进入默认层。
- 较重、较窄或适配性不稳定的思想，只能进入按需层。
- 听起来高级但收益低、容易误用、会拖慢、会污染规则层的思想，直接淘汰。

## Anti-pollution rule
任何导致错误答案、错误运行、明显低效、伪深刻感增强但任务表现无提升的思想，都应视为污染源而不是知识资产。

## Acceptance standard
不看说法是否高级，只看：
- 答案是否更准
- 路径是否更正
- 任务是否更快更稳
- 错误是否更少
- 系统是否更轻而不是更重
