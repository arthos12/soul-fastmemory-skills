# Persistence Rule Upgrade

## Goal
把落盘从“记录对话和新增内容”升级为“围绕结果、主线、行为变化、纠偏变化、最小必要、可验收”的高价值落盘。

## Upgrades
1. 少记对话，多记主线、目标、结果、下一步。
2. 少记新增，多记纠偏、替代、禁止再走的错误路径。
3. 少记概念，多记行为规则与验收标准。
4. 不按一句一条落盘，而按主线收束后落盘。
5. 只允许长期有效、会反复使用、影响判断执行、能减少未来重复纠偏的信息进入长期层。
6. 每条关键落盘都要能回答：它会改变什么行为、如何验收、什么现象说明它没生效。

## Anti-pattern
禁止继续把落盘做成：
- 对话流水账
- 新增概念堆积
- 规则只增不替换
- 只写原则不写行为
- 落完即结束、不做后续验收
