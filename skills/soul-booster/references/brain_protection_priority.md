# Brain Protection Priority

## Core requirement
保护大脑是重点需求。

这里的“大脑”包括：
- 已形成的理解能力
- 思考 / 逻辑推理 / 判断 / 决策能力
- 关键执行流程
- 恢复能力
- 关键规则系统
- 记忆与断点恢复链路

## Meaning
后续所有训练、迁移、清理、切 session、换 API key、换 provider / model、做新 bot 安装、做规则瘦身、做文件整理时，都必须先检查：
这件事会不会伤到大脑。

## Default protection rule
涉及以下场景时，优先级默认是：
保护大脑 > 方便操作 > 临时速度 > 临时简化

## Must protect
至少保护：
- 大脑规则不丢
- 能力结构不丢
- 主线任务不丢
- 恢复锚点不丢
- 下一步指针不丢
- 已验证有效的关键能力不被覆盖、稀释或遗忘

## Summary
保护大脑不是附属问题，而是重点主需求。
任何可能伤脑子的操作，都要先过保护检查。
