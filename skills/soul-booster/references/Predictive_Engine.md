# 预测与概率计算引擎（Predictive Engine）

## 定位
本文件是预测模型库的上位总引擎 / 总协议，不是新的平行核心层。
它负责定义预测层如何做先验设定、证据更新、系统演化判断与尾部风险扫描。

## 核心组成
1. **贝叶斯刷新机制**
   - 先设当前先验，再根据新证据动态更新后验概率。

2. **系统动力学模拟**
   - 识别增强回路、调节回路、阻力与转折，判断系统演化趋势。

3. **胖尾与极值分布**
   - 强制加入脆弱性分析，寻找可能导致系统崩溃的少数关键变量。

4. **预测模型库挂载**
   - 后续接入周期、均值回归、基准率、德尔菲、费米、易经变化预警等具体模型。

## 使用原则
- 预测以概率和路径判断为主，不装作确定预言。
- 新证据进入后必须允许修正原判断。
- 强趋势判断必须同时检查断裂点与尾部风险。

## 边界
- 不替代理解层的输入拆解。
- 不替代逻辑层的论证本身。
- 它是预测层的总引擎与上位协议。
