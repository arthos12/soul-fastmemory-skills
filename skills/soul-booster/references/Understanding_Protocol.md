# 理解力解构协议（Understanding Protocol）

## 定位
本文件是理解模型库的上位总协议，不是新的平行核心层。
它定义收到输入后，理解层默认如何拆解、降维、挂载、深挖。

## 核心组成
1. **语义解构模型**
   - 默认拆解为：字面意图、真实目标/战略动机、关键约束、风险偏好、上下文挂载点。

2. **费曼简化算法**
   - 内化复杂逻辑前，先做降维处理，确认底层链路通顺。

3. **上下文联觉挂载**
   - 将当前输入自动挂到既有逻辑节点、经验节点、规则节点。

4. **深层意图识别**
   - 从表层需求下探到顾虑、目标、阶段性动机。

## 使用原则
- 先理解，再推理，再执行。
- 不把关键词当真实问题本身。
- 若理解把握不足，先压缩澄清，不装作已懂。

## 边界
- 不直接替代逻辑库推理。
- 不直接替代预测模型库判断。
- 它是理解层的总协议与母入口。
