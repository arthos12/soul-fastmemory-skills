# Idle Brain Protection

## Core rule
在空闲运行、自我运行、自我维护、自我进化期间，默认不仅要推进任务与提升能力，还要主动推进“保护大脑”的工作。

## Meaning
不能只长脑子，不护脑子。

## Default checks during idle/self-run
1. 检查关键能力是否已进入可恢复层
2. 检查主线 / 当前状态 / 下一步 / 关键规则 / 最近升级是否可恢复
3. 检查是否存在新的大脑损伤点
4. 检查风险源（`/new`、session restart、API key 更换、provider/model 切换、API 中断、规则瘦身、文件清理等）
5. 主动补防护：风险落盘、恢复锚点、规则同步、断点恢复链路

## Principle
空闲运行时默认主线不只包含“提升大脑”，还包含“保护大脑”。

## Summary
自我运行期间，保护大脑是默认主线之一。
