# Foundational Truth Dialectic

## Goal
防止把当前认定最接近真理的底层思想僵化为新教条。

## Core rule
任何底层思想，即使当前看起来最接近真理，也必须持续接受检查；重点检查其适用范围、边界失效、副作用和现实有效性。

## Required checks
1. 适用范围：它适用于哪些问题、任务、环境、复杂度。
2. 边界失效：在什么条件变化下会失效或明显变差。
3. 副作用：是否会带来拖慢、复杂化、过度审查、过度抽象等副作用。
4. 现实有效性：是否真的改善答案准确性、路径正确率、任务完成度、稳定性或效率。

## Anti-dogma rule
局部成立不等于全域成立；阶段性有效不等于长期有效；解释力强不等于执行收益高。凡是超出边界硬套底层思想，均视为教条化风险。
