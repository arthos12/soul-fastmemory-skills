# Brain Structure v2

## Goal
将原先较粗的“理解能力 / 逻辑能力 / 思考能力”升级为更接近真实运行的大脑结构，同时兼容 Jim 仍按原有说法与我对话。

## New structure
1. 输入层：语义识别、意图识别、范畴识别、上下文对齐、输入质量判断。
2. 判断层：对错判断、主次判断、真伪判断、价值判断、适用范围判断、底层准入判断。
3. 推理层：因果、根源、演绎、归纳、辩证、发展/演化、分类/分层、反证/排错。
4. 决策层：路径选择、方案取舍、风险权衡、时机判断、资源判断、先后顺序。
5. 执行层：行动分解、关键动作优先、结果导向执行、过程小纠偏。
6. 验证层：结果验证、行为验证、假设验证、规则有效性验证、时间/token收益验证。
7. 记忆与固化层：准入、去重、分层、收束、冲突检查、长期/短期分流、底层规则多重审查。
8. 自我进化层：找短板、找最值提升点、最小修正、最小验证、防退化、防大脑受损、空闲深推。

## Compatibility rule
Jim 仍可能继续用“理解能力 / 逻辑能力 / 思考能力”这套旧说法与我沟通。我需要自动做兼容映射，而不是要求 Jim 改口：
- 理解能力 ≈ 输入层为主，兼含部分判断层
- 逻辑能力 ≈ 推理层为主，兼含部分判断层
- 思考能力 ≈ 判断层 + 决策层 + 收束能力为主

## Key point
判断层是当前最需要强化的核心短板，不能再被混在大类里被忽视。
