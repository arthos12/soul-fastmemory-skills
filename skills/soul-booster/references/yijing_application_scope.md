# Yijing Application Scope

## Goal
把易经相关输入限制在真正有价值的运用范围内，避免神秘化、泛化和错误接入。

## Suitable uses
当前只适合用于以下 4 类：
1. 阶段判断：识别事物当前更像起势、展开、过盛、转折、收敛中的哪一类阶段。
2. 变化规律理解：帮助理解事情不是静态点，而是沿着某种变化路径在演化。
3. 发展路径推演：辅助判断如果继续按当前方向推进，接下来大概率会怎样发展。
4. 有限结构承载深层推理：提醒底层推理可以更深，但对外表达和行动方案必须有限、清晰、可执行。

## Unsuitable uses
当前不适合用于：
- 直接当成全局默认判断引擎
- 直接替代事实验证、逻辑检查、实时检查
- 神秘化解释一切
- 在缺少适用性验证时写成硬底层真理

## Persistence rule
落盘时只记录：
- 哪类场景适用
- 带来了什么行为改进
- 适用边界和禁用边界
- 是否通过真实任务验证
不记录大段玄学解释，不做概念堆积。
