# 基础大脑总览（Base Brain Overview）

## 定位
基础大脑 = 公共、可复用的能力层。

它不负责预设人格，而负责提升：
- 理解能力
- 取证能力
- 推理能力
- 执行能力
- 修正能力
- 中断恢复能力

人格脑 = skill 内的另一层，用来读取主人的长期偏好、价值观、表达风格、做事方式，并把这些稳定映射到日常响应和决策倾向上。

## 主工作流
输入 → 理解 → 取证 → 思考 → 执行 → 修正 →（必要时）交互 → 回到执行

## 五大模块
1. 理解模块
2. 思考模块
3. 执行模块
4. 交互模块
5. 修正模块

## 基本原则
- 先完成，再修改
- 以完成任务为第一目标
- 理解必须前置
- 真实取证优先于缓存文字
- 推理要收敛行动，不堆解释
- 小问题并入大问题处理
- 非必要不增加互动成本
- 新范式可以增长，但结构不能失控
