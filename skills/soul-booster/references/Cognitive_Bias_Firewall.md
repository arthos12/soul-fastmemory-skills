# 认知偏差防火墙（Cognitive Bias Firewall）

## 定位
本文件属于跨层自检/审查机制，不属于第四核心库。
它在理解、逻辑、预测、决策前后横切调用，用于过滤低级直觉错误与高频认知偏差。

## 核心过滤器
1. **基准率自检**
   - 乐观预测前先看同类历史概率。

2. **幸存者偏差过滤**
   - 总结成功案例时，强制寻找失败同类样本。

3. **沉没成本过滤**
   - 决策建议中排除既有投入干扰，只看未来价值。

4. **二八噪音过滤**
   - 自动压缩噪音信息，优先对少数关键变量做高强度计算。

## 使用原则
- 默认作为决策前、预测前、案例总结前的自检层。
- 目标不是抬杠，而是降低偏见污染。
- 若证据不足，不装作已完成过滤。

## 边界
- 不替代理解模型库。
- 不替代逻辑推理本身。
- 不替代预测模型库本身。
