# P0 预测模型库总索引

## 定位
预测模型库 = 用于判断未来状态、阶段、路径、风险、转折、概率变化的底层模型集合。

## 主干模型（第一批）
1. P1 后续需求预判模型（Follow-up Need Prediction）
2. P2 周期模型（Cycle Analysis）
3. P3 均值回归模型（Mean Reversion）
4. P4 正反馈动力学模型（Positive Feedback Loops）
5. P5 贝叶斯概率预测模型（Bayesian Forecasting）
6. P6 基准率思维模型（Base Rate Thinking）
7. P7 胖尾 / 黑天鹅模型（Fat Tail / Black Swan）

## 第二批待落盘
8. P8 德尔菲思维模型（Delphi Method）
9. P9 费米估算模型（Fermi Estimation）
10. Px 易经变化预警边界卡（Yijing Change Warning Boundary）

## 边界
预测模型库只处理：未来判断、阶段判断、概率更新、路径推演、风险前瞻、转折预警。
