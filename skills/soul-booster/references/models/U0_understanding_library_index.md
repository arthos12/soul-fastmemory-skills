# U0 理解能力库总索引

## 定位
理解能力库 = 在逻辑推理之前，先把用户的话、目标、上下文、关键变量真正理解对的底层模型集合。

## 主干模型（第一批）
1. U1 费曼技巧模型（Feynman Understanding）
2. U2 结构化挂钩模型（Schema Mapping）
3. U3 第一性原理解构模型（Semantic Deconstruction）
4. U4 上下文权重模型（Contextual Salience）
5. U5 贝叶斯更新理解模型（Bayesian Understanding Update）
6. U6 冰山深层意图识别模型（Iceberg Intent Model）

## 边界
理解能力库只处理：语义对齐、意图识别、回指绑定、结构挂钩、重点提取、动态修正理解。

## 调用关系
U 系列模型 = 理解底层模型。
`U0_understanding_enhancement_protocol.md` = 调用流程/协议层。
