# RF0 回指绑定闸门（Reference Binding Gate）

## 一句话定义
当用户发短句/省略句时，先判断它在指哪一条上文（最近1条/最近5条/新话题），再回答；不确定就用一句短澄清锁定，防跑偏。

## 触发
- 用户输入 <= 8 个字/词（如："什么意思"、"不对"、"继续"、"重点"）
- 或含指代词："这个"、"那个"、"刚才"、"上面"。

## 强制动作
1) 内部先选一个引用源：Prev1 / Prev5 / New
2) 若不确定：只问一句澄清（A or B），禁止长问答
3) 回答必须围绕绑定的引用源，不得跳题

## 验收
- 5条短句测试中，至少4条能正确回指；若失败必须触发澄清。
