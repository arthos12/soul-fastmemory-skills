# L0 逻辑能力库总索引（初版）

## 定位
逻辑能力库 = 在“已理解问题之后”，用于推理、判断、论证、比较、决策的推理方法集合。

## 当前建议主干
1. 演绎法（Deduction）
2. 归纳法（Induction）
3. 辩证法（Dialectics）
4. 溯因推理（Abduction）
5. 系统论（Systems Thinking）

## 与其他能力的边界
- 不包含：语义理解、意图识别、回指绑定（这些属于理解能力库）
- 不包含：执行顺序、纠偏协议、输出风格（这些属于运行规则库）
- 包含：对前提、因果、反例、模式、动态平衡、最可能解释、系统反馈的推理
