# L0 逻辑能力库总索引（初版）

## 定位
逻辑能力库 = 在“已理解问题之后”，用于推理、判断、论证、比较、决策的推理方法集合。

## 当前建议主干
1. 演绎法（Deduction）
2. 归纳法（Induction）
3. 辩证法（Dialectics）
4. 溯因推理（Abduction）
5. 系统论（Systems Thinking）

## 边界
逻辑能力库只处理：对前提、因果、反例、模式、动态平衡、最可能解释、系统反馈的推理。
