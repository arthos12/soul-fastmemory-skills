## BR核心模式确认

### BR是"看到涨了才追"

```
不是: 提前判断 → 低位埋伏
而是: 看到涨 → 追进去
```

### 完整逻辑
1. 实时观察市场
2. 看到价格上涨 (0.3 → 0.5)
3. 确认趋势 (0.5 → 0.7)
4. 追进去买入 (0.7+)
5. 持有5分钟
6. 平仓
