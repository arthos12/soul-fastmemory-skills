# 人脉分类框架（Jim版）

## 核心原则
- 资源交换需要对等，不对所有人开放
- 嘴巴严 > 能力强，泄露比无能更伤
- 先验证，再深交，不要一上来全抛
- 信息分层，保护自己的圈子不被挖
- **找人脉永远找积极向上、寻求发展的人**
- **躺平的人、兼职的人、笨的人 → 直接放弃交往**
- **有破坏性的人也不能交往**

---

## 三层分类

### 类型一：深度交换型（最值得深交）

**特征：**
- 没有社区、没有自己圈子、没有建立小圈子
- 嘴巴严，不乱说
- 项目多，信息质量高
- 勤快，执行力强

**策略：**
- 深度绑定，资源互换
- 可以给核心信息
- 建立长期信任关系

---

### 类型二：业务交流型

**特征：**
- 有业务在跑，需要多看多学
- 保持沟通频率
- 主要是学对方在做什么项目

**策略：**
- 保持沟通，观察项目
- 可以给二手信息
- 不深度绑定

---

### 类型三：风险型

**特征：**
- 可能挖我的圈子
- 可能挖我的人
- 可能挖我的资源
- 动机不纯或边界不清

**策略：**
- 设防火墙
- 只给三手信息
- 不暴露核心关系网

---

## 操作原则

1. **验证优先**：不确认对方可靠之前，只给表象信息
2. **信息分层**：一手给深度交换型，二手给业务交流型，三手给风险型
3. **嘴巴严是硬门槛**：无论对方多强，如果嘴不严，直接降级
4. **防范意识**：定期检查有没有人试图挖你的圈子

|------|------|

| 杠杆 | 有的人能帮你放大10倍，有的人只能一起喝酒 |
